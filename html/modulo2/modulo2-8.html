La prueba para detección de VIH es obligatoria cuando se hacen donaciones de sangre, órganos, tejidos, semen y óvulos. Cualquier otra circunstancia es una decisión meramente voluntaria y exige el consentimiento de la persona involucrada para su realización.

Es recomendable realizar una prueba para detección de VIH si se desea:

  1. Iniciar una nueva relación amorosa.
  2. Tener hijos.
  3. Detectar si se vive con el VIH.
  4. Evitar un embarazo si las pruebas resultan positivas.
